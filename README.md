## :computer: 스터디 진행 방식
- 각 주차의 주제에 대한 개인 학습 및 정리
- 매일 1문제씩 풀고 Github에 커밋
- Pull Request를 통해 다른 팀원의 코드 리뷰 및 피드백
- 스터디 시간에 주제에 대한 이론 정리 및 발표, 질의 응답 진행
- 그 외 공유하고 싶은 문제, 풀리지 않은 문제가 있다면 스터디 시간에 함께 학습
